Rx for Javascript

Juha Paananen / Reaktor DevDay 2011-09-02
  • ~30 kb Javascript
  • Developed by Microsoft, originally for .NET
  • Free as in beer (not FOSS)
  • For event-driven (reactive) Javascript programming
  • Observable streams of events
  • Bindings to JQuery etc
  • Combinators (like FP list operations)
  • Removes explicit mutable state from complex apps
  • Good for input validation, calculation, games?
